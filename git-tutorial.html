O que é GIT?

É um sitema de controle de código.

Como o usar GIT no terminal?

  • Criar uma conta no GitHub
  • Criar um repositório no GitHub
  • Instalar o GIT
  • Localizar a pasta do arquivo do projeto no terminal bash - utilizar o comando cd
  • Digite o comando GIT INIT - este comando avisa o git para ficar de olho nesta pasta
  • Digite o comando GIT REMOTE ADD ORIGIN "link do repositório criado no GitHub"
  • Digite o comando GIT STATUS - mostra o que foi modificado
  • Digite o comando GIT ADD . - coloca tudo dentrode uma caixinha
  • Digite o comando GIT STATUS
  • Digite o comando GIT COMMIT -M "nome da etiqueta"
  • Digite o comando GIT PUSH ORIGIN MASTER"

Como trabalhar com o arquivo em outro computador?

  • Criar uma conta no GitHub
  • Criar um repositório no GitHub
  • Instalar o GIT
  • localizar onde deseja clonar o arquivo - utilize o comando cd
  • Digite o comando GIT CLONE "web url do repositório"
  • Digite o comando CD "nome da pasta"
  • Verifique o status digitando o comando GIT STATUS
  • Digite o comando GIT CHECKOUT -B GIT
  • Digite o comando GIT STATUS -ON BRANCH GIT
